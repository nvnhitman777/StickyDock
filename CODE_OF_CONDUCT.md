# Code of Conduct

StickyDock is an open-source project. We expect all contributors to be
respectful, welcoming, and considerate.

## Our Standards

- Be respectful and inclusive.
- Use welcoming language.
- Assume good faith and give people the benefit of the doubt.
- Avoid demeaning, discriminatory, or harassing behavior.
- Accept constructive criticism gracefully.

## Reporting Issues

If you encounter unacceptable behavior, please report it to the project
maintainer by opening an issue or contacting the repository owner.

## Enforcement

The maintainers are responsible for enforcing this Code of Conduct. Violations
may result in removal from the project or other actions as needed to maintain a
safe and welcoming community.

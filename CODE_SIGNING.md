# Code Signing Policy

StickyDock is an open-source Windows desktop application.

Official releases may be code-signed using an approved open-source signing
service such as SignPath Foundation.

## Roles

- Maintainer: nvnhitman777
- Reviewer: nvnhitman777
- Release approver: nvnhitman777

## Privacy

StickyDock does not transmit user information to external services unless the
user explicitly initiates that action.

## Build process

The local build creates an unsigned MSI. Official release artifacts may be
signed separately after the build completes.
